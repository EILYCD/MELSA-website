Credit-Saver Workflow

Follow this process to ensure zero waste of Cloudflare build credits. Develop locally, test on staging, and only deploy perfection.

1

Local Development

Download the repo to your Chromebook. Use VS Code to edit and preview files using `file://`. Ensure images load correctly.

Cost: 0 Credits
2

UAT Deployment

Push changes to the `uat` branch. Cloudflare automatically builds to `uat.melsawellness.pages.dev`. Test mobile responsiveness here.

Cost: 1 Credit
3

Production Release

Merge `uat` into `main` via Pull Request. This updates the live `melsawellness.ca` site.

Cost: 1 Credit
