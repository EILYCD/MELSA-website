Tech Stack Composition

A lightweight, high-performance static stack deployed on the edge.

  • HTML5 (40%): Semantic structure & content.
  • Tailwind (30%): Utility-first styling (CDN).
  • JS (20%): Logic for forms, nav, & i18n.
  • Backend (10%): Google Apps Script automation.
